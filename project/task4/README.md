# Реализация игры BlackJack на Python

## Описание:

Реализована игра в blackjack по основным правилам. В игре участвуют дилер и три бота:
- консервативный (играет осторожно),
- агрессивный (играет более рисково),
- оптимальный (с простой и более менее оптимальной математической стратегией).

Каждый бот в начале игры получает по 500 фишек. Раунд заканчивается, когда все игроки и дилер взяли карты и остановились.
Победитель определяется по общеизвестым правилам игры.
Игра заканчивается, когда закончились все раунды (в демонстрации число раундов ограничено пятью) или когда не осталось активных игроков (все обанкротились).

## Структура проекта:

Для удобства выделено несколько отдельных файлов для определения классов:
Card.py - игровая карта
Deck.py - колода карт
Hand.py - рука игрока
Players.py - сами игроки (общий класс, бот и дилер)
Strategies.py - стратегии игры ботов
blackjack.py - файл с основной реализацией игрового процесса.
main.py - файл для запуска демонстрации.

Реализованы тесты для каждого файла, максимально полно покрывающие функциональность.
